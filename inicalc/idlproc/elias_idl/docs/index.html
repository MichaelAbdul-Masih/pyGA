ELAIS IDL Libraries

Below are links to the Hypertext help facilities for IDL routines. These routines were developed as part of the ELAIS Project.

Some of the more obscure routines have been removed. It is quite possible that routines that remain do not work without routines that are have not been included. If you find any such missing routines or other problems, please email S.Oliver@Sussex.ac.uk

Some routines are modifications of routines written by others, in particular the Astrolib library. Our routines may fix bugs (some bugs that have since been fixed) or may give additional functionality (which may be highly specific to our purposes), we hope that we have made this clear in the headers, and hope we have not infringed any copyrights!

All code is supplied as is and comes with no guarentees. Although if you find any bugs we would like to know (please email S.Oliver@Sussex.ac.uk)

A tar file of all these routines is also available for export.

  • Astrolib Versions of astrolib routines
  • Astrom Local astrometry routines e.g. postage
  • Associate Local association routines
  • Graphics Graphics routines
  • General General routines: e.g. Model SEDs, Number Counts
  • Statistics Routines for doing stats on data e.g. noise, sky-flats
